# Online Bookstore SQL Analysis

## Project Overview
This project focuses on analyzing an online bookstore dataset using SQL to gain insights into book sales, customer behavior, revenue trends, and inventory status.

## Dataset Description
The dataset consists of three main tables:
- Books: Contains book details such as title, author, price, genre, and stock.
- Customers: Stores customer information including name and contact details.
- Orders: Records order transactions with quantity, order date, and customer-book relationships.

## Analysis Performed
- Analyzed total revenue and sales performance of books
- Identified top-selling books and high-value customers
- Evaluated customer purchasing behavior
- Checked inventory levels and remaining stock after sales
- Used time-based analysis to study order trends

## SQL Concepts Used
- Database and table creation with constraints
- Primary and foreign keys
- Joins (INNER, LEFT)
- Aggregate functions (SUM, COUNT, AVG)
- Subqueries
- Common Table Expressions (CTEs)
- Window functions (DENSE_RANK, LAG)

## How to Use
1. Create the database and tables using the schema.
2. Import the dataset into the tables.
3. Run the analysis queries from the SQL file to explore insights.
